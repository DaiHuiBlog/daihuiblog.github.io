工作流

你的本地仓库由 git 维护的三棵“树”组成。
第一个是你的工作目录，它持有实际文件；
第二个是暂存区（Index），它像个缓存区域，临时保存你的改动；
最后是 HEAD，它指向你最后一次提交的结果。
